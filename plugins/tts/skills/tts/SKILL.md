# TTS

Control automatic TTS playback of Claude Code responses.

## Usage

Use the MCP tools:
- `tts_on` — Enable TTS playback
- `tts_off` — Disable TTS playback
- `tts_status` — Check current TTS state
- `tts_stop` — Stop the current TTS playback
